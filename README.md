# Node Notes

Ultra simple list manager using backbone, express, and mongodb. 

## Installation

#### Get the app

```
git clone git@github.com:danlash/node-notes.git
npm install
```

#### Install mongo

```
start http://www.mongodb.org/downloads
```

## Run


```
mongod
node app.js
start http://127.0.0.1:3000/plugin.html
```
